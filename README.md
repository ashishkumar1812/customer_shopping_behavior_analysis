🛒 Customer Shopping Behavior Analysis

📌 Project Overview
This project analyzes customer shopping behavior to uncover patterns, trends, and key drivers behind purchasing decisions. It combines Python, SQL, and Power BI to transform raw data into meaningful business insights.

🎯 Objectives
Understand customer purchase patterns
Identify high-value customer segments
Analyze factors influencing buying decisions
Provide actionable business recommendations

🧰 Tools & Technologies
Python (Pandas, NumPy) – Data cleaning & preprocessing
SQL – Data analysis & querying
Power BI – Data visualization & dashboard
GitHub – Project management & version control

🚀 How to Use
1. Clone the repository
2. Run Python scripts for data preparation
3. Execute SQL queries for analysis
4. Open Power BI file to explore dashboard
